Capstone Project

CareConnect Medical Appointment Booking System

CareConnect is a comprehensive medical appointment booking system designed to streamline interactions between patients, doctors, and administrators.

Project Overview

CareConnect allows patients to schedule appointments, doctors to manage their schedules, and administrators to oversee the entire system efficiently.


Key Features

Patient Features

  • Secure authentication for patients.
  • Easy scheduling and cancellation.
  • View and edit profile details.

Doctor Features

  • View appointments and patient info.
  • Manage patient schedules.
  • Update availability and details.

Admin Features

  • View doctors, patients, and appointment stats.
  • Add and manage doctor profiles.
  • Monitor and cancel appointments.

Technologies Used

CareConnect is built using MongoDB, Express, React, and Node.js (MERN Stack) to provide a seamless and efficient experience.


Project Vision

The goal of CareConnect is to enhance patient-doctor interactions by providing a user-friendly and efficient online appointment booking system.
